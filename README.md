# Simpell
A simple proof-of-concept programming language built to learn interpreter fundamentals, such as tokenization, tree traversal, and Abstract Syntax Trees (ASTs).
Completed features:
* Functions
* If/Else
* Loops
* Expressions, including parenthetical sequences
* Numbers & math
* Strings
* Printing/random numbers
* Basic error handling (parsing & runtime)

A compiled .exe can be found in the /code folder. Some example code is given in /examples.

The future?
* Arrays
* Advanced string modification
* Functions as first-class citizens
